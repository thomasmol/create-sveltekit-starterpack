# Easy create new prod ready sveltekit projects

Defaults:
* Typescript
* Prettier
* Eslint
* Playwright
* Vitest
* Some icons (loading, external link)

Options for:
* TailwindCSS
* AuthJS (wip)
* Prisma (wip)
* OpenAI (wip)
* EthersJS (wip)
* @solana/web3.js (wip)


```bash
 npm create starterkitkit my-app
```
